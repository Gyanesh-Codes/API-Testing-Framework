# Screenshot Evidence Instructions

Add real execution evidence here after running the collection in Postman or Newman.

Recommended screenshots:
- Imported Postman collection with folder hierarchy visible.
- Successful Login request showing 200 response and token assertion.
- Failed Login request showing 400 response and error assertion.
- Collection Runner summary for Smoke Testing.
- Collection Runner summary for Regression Testing.
- Bug evidence screenshot for invalid payload or empty body behavior.

Naming convention:
- `01_collection_import.png`
- `02_successful_login.png`
- `03_failed_login.png`
- `04_smoke_runner_summary.png`
- `05_regression_runner_summary.png`
- `06_bug_evidence_invalid_payload.png`
